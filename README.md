# FAQ Chatbot Using NLP

## Project Overview

This project is an FAQ Chatbot built using Python, TF-IDF Vectorization, and Cosine Similarity. It answers user questions by matching them with the most relevant FAQ.

## Files Included

* chatbot.py - Main chatbot logic
* app.py - Streamlit web interface
* faq_data.json - FAQ dataset
* requirements.txt - Required libraries

## Installation

python -m pip install -r requirements.txt

## Run Terminal Version

python chatbot.py

## Run Web Interface

python -m streamlit run app.py

Then open:

http://localhost:8501
